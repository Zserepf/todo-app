# User Personas

## Persona 1: Alex — The Busy Professional

- **Role**: Primary todo app user
- **Goals**: Stay on top of deadlines, never miss important tasks
- **Behaviors**: Creates todos with due dates, checks app multiple times per day, wants proactive reminders
- **Pain Points**: Forgets tasks unless reminded, overwhelmed by long lists, misses deadlines
- **Technical Comfort**: Moderate — expects modern UI patterns (badges, dropdowns, real-time updates)

## Persona 2: Sam — The Team Lead

- **Role**: Power user managing many todos
- **Goals**: Track team deliverables, set reminders for follow-ups, quickly triage notifications
- **Behaviors**: Has 50+ active todos, sets reminders for future check-ins, bulk-manages notifications
- **Pain Points**: Notification fatigue if too noisy, needs "mark all read" for efficiency
- **Technical Comfort**: High — comfortable with datetime pickers, understands notification patterns
